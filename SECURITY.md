# Security Policy

## Scope

This repository is a **knowledge base** (Markdown files, C# templates, shell scripts). It does not contain application code that processes user data.

The only executable component is the **NPX installer** (`bin/install.mjs`), which copies files to the user's project directory.

## Reporting

If you find a security issue in the installer script, please report it via [GitHub Issues](https://github.com/niaka3dayo/agent-skills-vrc-udon/issues).

## npm Provenance

Published packages include [npm provenance attestation](https://docs.npmjs.com/generating-provenance-statements) for supply chain verification.
